# Deployment Guide

## 1. Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit"

git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/hellogrok.git
git push -u origin main
```

## 2. Quick Demo (See Hello World)

Deploy to **Railway** or **Render** directly from GitHub for the fastest way to see the running application.

See the main README for the endpoint: `/api/greetings/hello`
